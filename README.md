Задача подготовить Dockerfile для сборки докер образа.
Запущенный из данного образа контейнер должен уметь принимать
ASCII строку на вход и в качестве результата печатать сумму ASCII
кодов всех символов, входящих в строку ▪ Запуск контейнера должен
выполняться командой docker run util somestring. docker run util aaa
должно вывести 291.
